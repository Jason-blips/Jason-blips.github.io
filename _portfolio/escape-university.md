## Description

Built a 2D campus-themed adventure game with tile-based maps, character animations, and interactive collectibles. The game features an immersive campus environment with dynamic gameplay mechanics.

## Key Features

* Tile-based map system using Tiled Maps for level design
* Character animations and interactive collectibles
* Dynamic lighting system using frame buffer rendering for torch mechanics and environmental effects
* AI companion system with pathfinding algorithms and behavior-state management
* Cross-platform deployment pipeline for desktop (LWJGL3) and web (GWT) platforms

## Technical Highlights

* Game Engine: libGDX framework for cross-platform game development
* Map Design: Tiled Maps for efficient level creation
* Graphics: Frame buffer rendering for advanced lighting effects
* AI: Pathfinding algorithms for companion behavior
* Deployment: Multi-platform support (Desktop and Web)

## Description

Built a COVID-19 health check-in web application for daily health status and location tracking. The application provides a comprehensive solution for monitoring health status during the pandemic.

## Key Features

* Integrated Baidu Maps API with geocoding and autocomplete for interactive location selection
* Developed responsive UI using Element UI with forms, tables, pagination, and data visualization
* Designed RESTful API endpoints for authentication, health record submission, and data retrieval
* Created admin dashboard with advanced filtering and pagination for comprehensive record management
* Configured Webpack build pipeline with separate dev/prod environments for optimized deployment

## Technical Highlights

* Frontend: Vue.js framework with Element UI component library
* Maps Integration: Baidu Maps API for location services
* Build Tools: Webpack with environment-specific configurations
* API: RESTful architecture for backend communication
